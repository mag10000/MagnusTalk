Welcome to magnus talk your trusty dialogue plugin

Instalation
---------------
>To download go to this [link](https://github.com/mag10000/MagnusTalk/archive/37bd2dc45b368ba1040005a5cfe01d75a04f440d.zip).

>Unzip the newly dowloaded file.

>Paste the **addons** folder from the unzipped file into you godot project.

>Go into: **project settings > plugins**.

>Enable the **MagnusTalk** plugin.

>If you see a Magnus Talk popup then the plugin has been installed, if you don't restart the editor.

Documentation
---------------
You can either use the documentation built in with the plugin or you can use the documentation [here](https://magnusknisely.wixsite.com/magnus-talk-plugin/doc-to-be-viewed).
